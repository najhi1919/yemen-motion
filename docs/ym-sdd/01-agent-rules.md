# Agent Rules

- Do not modify any file outside the task's Allowed Files.
- Do not use `git add .`.
- Do not use `git reset`, `git restore`, or `git clean`.
- Do not read or print secrets.
- Do not modify Auth without an explicit task.
- Do not modify frontend and backend in the same task unless explicitly allowed.
- Do not add libraries without explicit approval.
- Do not execute more than one task at a time.
- Run the checks required by the task.
- Write the final report in Arabic.
- Git is the source of truth, not the agent report.
